### Hi, I'm Koishore 👋

#### I build the deterministic, trustworthy layer around AI agents — authorization, audit, testing, and control.

Most of my recent work circles one idea: **keep determinism in charge, and let the LLM do only the genuinely open-ended part.** Building since 2016 — from CS at Ashoka University through a run of products and startups, now focused on AI-agent infrastructure.

### 🚀 Featured

- 🛡️ **[delego](https://github.com/Delego-Dev/delego)** — a policy & audit *firewall* for AI-agent actions. Answers what credential brokers can't: *is this action what the human actually authorized?* Deterministic policy (no LLM in the decision path), action-bound human approval, and a tamper-evident Ed25519-signed audit chain.
  `Python` · `MCP` · Apache-2.0 · CI 3.10–3.12
- 🎙️ **[voice-agentic-ai-assistant](https://github.com/koishore/voice-agentic-ai-assistant)** — a voice assistant that routes intent by deterministic regex and calls an LLM *only* when the request is truly open-ended. 105 tests passing, cross-session memory, transparent multi-provider fallback (Groq → Gemini → Cerebras), offline Whisper STT.
  `Python` · `Chainlit` · `LiteLLM`
- 🧪 **[testzeus-mcp](https://github.com/koishore/testzeus-mcp)** — an MCP server that exposes a real end-to-end testing engine to Claude Desktop / Cursor / VSCode, so you author and run E2E tests in plain English.
  `Python` · `FastMCP` · `TypeScript`
